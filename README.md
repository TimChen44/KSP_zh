# KSP_zh
《坎巴拉太空计划》汉化项目

#### 官方中文版的已经推出，一个时代的终结，预示着更美好的时代已经到来。

### 支持的游戏版本
v1.2.2.1622 32位

注：不同版本游戏不支持

###配置文件相关介绍
请详见上一版本的介绍[KSP 0.24.2汉化包技术解密](http://bbs.deeptimes.org/forum.php?mod=viewthread&tid=2137227&extra=page%3D2)

将resource文件夹中的三个文件夹覆盖游戏根目录中同名的三个文件夹。
